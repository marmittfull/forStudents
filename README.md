### Installation

1. Clone the repository
   ```sh
   git clone https://github.com/marmittfull/forStudents.git
   ```
2. Install repository packages
   ```sh
   yarn
   ```
3. Run the following code on the project root directory
   ```sh
   yarn start
   ```
